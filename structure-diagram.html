Comandos para começar
1 — Instalar dependências
$ cd logos
$ npm install
instala frontend + backend via workspaces
2 — Configurar ambiente
$ cp backend/.env.example backend/.env
$ cp frontend/.env.example frontend/.env
edite os arquivos .env com seus dados
3 — Banco de dados
$ createdb logos_dev
$ npm run db:migrate
$ npm run db:seed
4 — Rodar em desenvolvimento
$ npm run dev
Frontend: localhost:5173 · API: localhost:3000
Docs: localhost:3000/docs (Swagger)
